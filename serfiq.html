SER_FIQ

 SER_FIQ (gpu:int=0, det:int=0)

Reimplementing Insightface’s FaceModel class. Now the dropout output and the network output are returned after a forward pass. Parameters ———- gpu : int, optional The GPU to be used by Mxnet. The default is 0. If set to None, CPU is used instead. det : int, optional Mtcnn option, 1= Use R+0, 0= Detect from beginning. The default is 0. Returns ——- None.

Type Default Details
gpu int 0 Which gpu should be used -> gpu id
det int 0 Mtcnn option, 1= Use R+O, 0=Detect from beginning
